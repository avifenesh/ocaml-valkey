Module Valkey.Json

Typed helpers for the Valkey JSON module.

The API keeps JSON payloads as serialized strings instead of depending on a particular OCaml JSON library. Callers can use Yojson, jsonm, ppx_deriving_yojson, or hand-written encoders at the boundary and pass the resulting JSON text here.

Commands route by document key, so they work correctly in cluster mode as long as multi-key calls use co-located keys when Valkey requires it.
